Descargars e Instalaciones:
___________________________

	_primero debemos instalar el servidor local Xampp


	_luego debemos descargar el workpress, en workpress.org


	_luego debemos descomprimir el workpress, que descarcamos y crear un carpeta de proyecto

	en la ruta del servidor local Xampp/htdocs/name_proyect, y pegar allí los archivos extraidos

	del workpress




-------------------------------------------------------------------------------------------------------

Bases de Datos:
_______________



	_luego debemos crear la base de datos para workpress, para ésto, vamos al panel de control

	"XAMPP Control Panel", y vamos a la sección de "MySQL/Conf"


	_y dentro del panel de configuración vamos a la sección de "Bases de datos", y creamos un base

	de datos con el nombre que queramos, indicando un nombre para ésta un cotejamiento, ej

	"nombre: ipm", "Cotejamiento: utf8mb4_unicode_ci", luego le damos ha crear



	wp-config-sample:
	_________________


	_ahora debemos direccionar la base de datos que hemos creado con nuestro workpress, para ésto,

	vamos ha Xampp/htdocs/ej:ipm/", y abrimos el archivo "wp-config-sample.php", en nuestro editor

	ej, brackers, ó sublime text, y en:


	_define('DB_NAME', 'databse_name_here');

		_quedando ej: define('DB_USER', 'ipm');


	_define('DB_USER', 'username_here');

		_quedando ej: define('DB_USER', 'root');


	_define('DB_PASSWORD', 'password_here');

		_define('DB_PASSWORD', '');


	_define('DB_HOST', 'localhost');


		_la dejamos tal cuál, está


	_por último "guardamos como", el archivo y lo renombramos "wp-config.php", luego podremos

	borrar el archivo de ejemplo "wp-config-sample.php"



-------------------------------------------------------------------------------------------------------

Instalar y Configurar Workpress:
________________________________


	_luego tengamos agregado workpress, a la estrutura de archivos de xampp, y hayamos creado la base

	de datos y relacionado ésta con workpress, ahora podremos abrir workpress, a través, de su ruta

	de acceso para proceder a configurarlo


	_para ésto, vamos a su ruta ej: "http://localhost/ej:ipm/", así se abrirá el proyecto de workpress,

	cambiando la ruta ha "http://localhost/ej:ipm/install.php"
